Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre o termos para descobrir qual tag devetrautilizar em cada caso.

  • Paragrafo

    • Ex.: <p>Assim se escreve um paragrafo em HTML5</p>

  • Titulos

    • Ex.: <h1>Assim se insere um Titulo em HTML5</h1>

    Os titulos vao de h1 ate h6, onde a variacao e no tamanho da fonte.

  • Imagens

    • Ex.: <img src="" alt="">

    A tag <img> nao tem tag de fechamento. Onde src e o local da imagem (que pode ser local, ou hospedado em site externo) e alt e o texto alternatico, caso a imagem nao possa ser exibida.

  • Abreviacoes

    • A tag <abbr> e usada para explicar abreviacoes.faz surgir uma caixinha com o significado.

    <abbr title="Explicacao da abreviacao">Abreviacao</abbr>

  • Codigos-fonte
  • Listas numeradas

    • Tag usada para criar listas com ordem. Onde pode-se usar numeros (1, 2, 3...), numeros romanos maiusculos ou minusculos (I, II, III...) (i, ii, iii...) letras maiusculas ou minusculas (A, B, C...) (a, b, c...)

    Ex.: <ol>Elementos a serem ordenados. Deverao conter a tag <li> entre cada item da lista</li> para que haja a quebra de linha. </ol>

    Para definir a forma de contagem, usa-se: <ol type=""> entre as aspas: "1" para numerais, "A" para letras maiusculas, "a" para letras minusculas, "I" para numeros romarnos maiusculos e "i" para numeros romanos minusculos.

  • Listas com demarcadores

    • Tag usada para criar listas que nao precisem de ordem.

    Pode-se alterar o marcador usando: <ul type="">

    as formas de marcadores aceitos sao: "disk", "circle", "square"

  • Ancora

    • Ex.: <a href="https://google.com">Google</a>

    Para abrir o hiperlink em uma nova aba do navegador, usamos:<a href="https://google.com" target="_blank">Google</a>
